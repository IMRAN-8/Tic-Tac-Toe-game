# Tic Tac Toe Game

A modern, responsive Tic Tac Toe game built with HTML, CSS, and vanilla JavaScript. It supports two-player gameplay, score tracking, draw detection, and alternating starting turns between X and O.

## Features

- Two-player Tic Tac Toe gameplay
- Automatic win detection
- Draw detection
- Score tracking for Player X, Player O, and draws
- Alternating first turn each round: X starts first, then O starts the next round
- New Round and Reset Score controls
- Clean, responsive UI
- Built without external libraries

## Tech Stack

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/IMRAN-8/Tic-Tac-Toe-game.git
```

2. Open the project folder:

```bash
cd Tic-Tac-Toe-game
```

3. Open `index.html` in your browser.

No installation or setup is required.

## Gameplay Rules

- Player X starts the first round.
- Player O starts the second round.
- Starting turns continue alternating after each new round.
- The first player to place three marks in a row, column, or diagonal wins.
- If all cells are filled without a winner, the round ends in a draw.
